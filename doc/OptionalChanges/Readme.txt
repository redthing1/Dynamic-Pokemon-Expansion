This folder and its subfolders contain the various files that you can edit to configure DPE for your own needs!
It is recommended that you go in this order:

Edit what is contained in the Include Folder
Edit what is contained in the Src Folder
Edit what is contained in the Strings folder
Add or remove cries and graphics based on your edited defines
Edit special_inserts.asm


Here is a list of files you can edit easily, and others you probably shouldn't touch unless you knew what you were doing.
